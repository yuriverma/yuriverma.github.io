Project Overview

Built for a Bank of Baroda use case, this intelligent chatbot revolutionizes credit card distribution by simulating human-like interactions and providing personalized financial guidance. The system leverages advanced NLP and machine learning to understand complex user queries and deliver tailored product recommendations.

Developed as part of a Founder's Office internship assignment, this project showcases the potential of conversational AI in transforming traditional banking experiences through intelligent automation and personalization.

Key Features & Capabilities

💬 Contextual Conversational Flow

Maintains sophisticated conversation state across multiple topics, dynamically adjusting responses based on user intent even when users switch topics mid-session.

Extracts precise answers from preloaded FAQs and SOP documents using advanced vector search and semantic matching algorithms for accurate information retrieval.

Custom rules engine extracts upselling triggers from user responses and intelligently suggests better alternatives or complementary add-ons.

Challenges & Solutions

🧩 Context Switching

Challenge: Users frequently change topics mid-conversation, losing context.

Solution: Implemented state management with LangChain memory modules to maintain conversation history and seamlessly handle topic transitions.

📊 Personalization Accuracy

Challenge: Generic responses without considering user financial profile.

Solution: Integrated CRM-style inputs with LLM prompting to generate tailored recommendations based on income, age, and credit score.

🔍 Knowledge Retrieval

Challenge: Accurate extraction from extensive FAQ and policy documents.

Solution: Implemented vector-based semantic search using ChromaDB for precise information retrieval with 95% accuracy.

⚡ Response Speed

Challenge: LLM processing causing delays in user interaction.

Solution: Optimized pipeline with parallel processing and caching, achieving 3ms average response time.
